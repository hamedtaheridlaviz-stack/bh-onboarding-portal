# Betterhomes Supabase Onboarding V2

Upload all files in this folder to the ROOT of your GitHub repo:
- index.html
- client.html
- styles.css
- config.js
- app.js
- client.js
- supabase-schema.sql

Then Vercel will redeploy.

In Supabase:
1. SQL Editor > run supabase-schema.sql
2. Storage > confirm bucket exists: onboarding-documents, private

Do not use or upload any sb_secret key.
